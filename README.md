# Claude Code Teams VS Code Extension

This extension opens a Claude Code Teams-like panel with 10 agent tabs, each with chat and code area, and integrates with VS Code LLM (Copilot/CodeGPT) for code generation. No web browser required; all UI is inside VS Code. Includes command to launch the panel, agent handoff logic, and basic LLM prompt integration.

## Features
- Command to launch the Claude Teams panel
- 10 agent tabs (one per agent)
- Each tab has chat and code area
- Agent handoff logic (auto or manual)
- LLM prompt integration (Copilot/CodeGPT)

## Getting Started
1. Open the command palette (Ctrl+Shift+P) and run `Claude Teams: Open Panel`.
2. Interact with agents, submit ideas, and watch collaborative code generation.

## Requirements
- VS Code 1.70+
- Copilot or CodeGPT extension (for LLM integration)

## Development
- All code is in the `src/` directory.
- See `copilot-instructions.md` in `.github/` for project automation and checklist.
